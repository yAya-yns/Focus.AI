# Focus.AI 
<h1>**Overall**<h1/><br/>
This webcam-based eye tracking system based on the python library *NumPy*, *OpenCV* and *Dlib*. It gives you the exact position of the pupils direction and the location where the eyes focus on.<br/>
<br/>
